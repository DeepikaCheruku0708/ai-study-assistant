# AI Study Assistant

A beginner-friendly AI application built using Python and Streamlit.

## Features

- Enter a study topic
- Generate study notes
- Easy-to-use web interface

## Technologies Used

- Python
- Streamlit
- Git
- GitHub

## How to Run

1. Install Streamlit

pip install streamlit

2. Run the application

streamlit run app.py
